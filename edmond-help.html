Collections

Collections provide the main folder structure and are required to upload and organize items. All items in a collection are described by a shared set of metadata elements, as specified by the metadata profile for the particular collection.


Collection List

The list displays all collections accessible to the user, including: all collections created by or shared with the user (identified via the user account) and all collections released by other users. If the user is not logged in, the list contains released collections only.

In the Collection List, users can change the number of hits displayed per page. Further on, the sorting can be modified and following filters can be applied:

  • All (except discarded) = all collections visible to the user, except discarded collections
  • My (except discarded) = all collections created by or shared with the user, except discarded collections
  • Only private = all collections in the state "private", created by or shared with the user
  • Only published = all collections in the state "released"
  • Only discarded = all collections in the state "discarded", created by or shared with the user

The status of the collection is indicated by the icon right to its name: Private collections are tagged with a lock, discarded collections with a trash bin and released collections are not accompanied by an icon.


Create a Collection

Note: Users require the appropriate authorization to create new collections. Please contact the support team with any question regarding your account.

  • Go to + New and click on Create collection.
  • Enter the required information (collection name, author name and organization). Information on further authors and a description of the collections content can be added optionally.
  • Via the plus (and minus) buttons, further fields for authors and organizations can be added (or removed).
  • Decide to define a metadata profile later or select an existing profile.
  • Click on Save.

Note: The newly created collection is automatically set to state "private".


Create a Metadata Profile

  • Go to Collections.
  • Select Metadata Profile for the corresponding collection.
  • Click on Add a metadata profile on the right hand side.
  • Choose weather you want to create a new or select an existing metadata profile.
  • When you choose to define a new metadata profile, please provide a title and a description for the metadata profile. In addition, you will need to define the specific set of metadata elements for the collection. Each element consists of:
    • a data type (e.g. person, date, number, text)
    • one or more labels (please choose the plus button to add labels for other languages)
    • [optionally] an online available vocabulary the element is linked to
    • [optionally] a list of predefined values for the element
    • the setting "Multiple" (to be set if an element can occur more than once for an item)
    • the setting "Preview" (to be set if an element should be displayed at mouse over in the item list)
  • When you choose to select an existing metadata profile, please select an entry from the list of available profiles. Afterwards, you have the choice to reuse the exact same profile or to use the selected metdata profile as a template.
  • Click on Save or Select.

Manage Items in a Collection

Upload Items to the Collection:

  • Go to Collections.
  • Select Upload content for the corresponding collection.
  • Add files via selecting them on your system and then start the upload.
  • Click on Edit uploaded images at the bottom of the screen to start assigning metadata.

Delete /Discard Items:

  • First you have to consider the state of your collection.
  • If your collection is private, you can delete items within the collection.
    • Go to Collections.
    • Click on View items for the corresponding collection.
    • Select the items you want to delete.
    • Go to Selected and click on Delete selected items.
    • If you want to delete all items: Go to Action and click on Delete all items.
  • If your collection is released, you can only discard items within the collection.
    • Go to Collections.
    • Click on View items for the corresponding collection.
    • Select the items you want to discard.
    • Got to Selected and click on Discard selected items.
    • If you want to discard all items: Go to Action and click on Discard all items.
    • Enter a comment explaining why the item has been discarded.
    • Discarded items are not listed and cannot be retrieved in the system any longer. Some basic metadata and the discard comment will be displayed if the item is requested via its URL.

Assign /Edit metadata values:

  • Go to Collections.
  • Click on View items for the corresponding collection.
  • Select the items you want to assign metadata to.
  • Go to Selected and click on Edit selected.
  • If you want to edit all items: Go to Edit and click on Edit all items.
  • On the next screen:
    • Select a metadata element from the drop-down list.
    • Either assign the same metadata value to all selected items via the work area at the top. Select if the new entry should overwrite existing values and click on Add to all.
    • Or add values directly to each item separately.
    • Click on Save.
  • To modify the metadata for one item only, click on it. Then go to Edit and click on Edit metadata.

Edit the Metadata Profile

  • Go to Collections.
  • Click on Metadata Profile for the corresponding collection.
  • Go to Edit and click on Edit profile.
  • To change the profile used by the collection, go to Action and click on Change your metadata profile.

Edit the Collection Metadata

  • Go to Collections.
  • View the collection by clicking on the name of the collection.
  • Go to Edit and click on Edit collection metadata.
  • On the next screen:
    • Modify the values you wish to update.
    • Notifications: Tick this option to receive an email if a file is downloaded from the collection.
    • Upload Logo: Here you can upload an image file which will be used (if available) as preview picture for the collection in the collection list.
    • Click on Save.

Share a Collection

Note: Collections can only be shared with existing users or user groups. Please contact the support team to request additional accounts or groups.

  • Go to Collections.
  • View the collection by clicking on its name.
  • Go to Share and click on Share collection.
  • Decide to share the collection with an individual user or a user group.
  • Enter the email address of the user or select the user group.
  • Choose the grants you want to assign to the user or the user group.
  • Click on Save.

Release a Collection

Note: You can only release private collections. A release makes the collection and all included items visible to the public. Afterwards neither the collection nor the items can be deleted any longer, but they need to be discarded instead. The release does not freeze the collection completely, in particular new items can still be added.

  • Go to Collections.
  • View the collection by clicking on its name.
  • Go to Action and click on Release collection.
  • The collection is now released and visible to the public.

Delete /Discard a Collection

  • First you have to consider the state of your collection.
  • If your collection is private, you can delete your collection.
    • Go to Collections.
    • View the collection by clicking on its name.
    • Go to Action and click on Delete collection.
    • The collection and all included items are now deleted.
  • When your collection is released, you can only discard your collection.
    • Go to Collections.
    • View the collection by clicking on its name.
    • Go to Action and click on Discard collection.
    • Enter a comment explaining why the collection has been discarded.
    • The collection is now discarded. That means the collection and all included items are not listed and cannot be retrieved in the system any longer. Some basic metadata and the discard comment will be displayed if the collection is requested via its URL.

Assign a DOI to a Collection

Note: You can only assign a DOI to a released collection and you can only assign one DOI per collection.

  • Go to Collections.
  • View the collection by clicking on its name.
  • Go to Action and click on Add DOI to collection.
  • Now you can decide whether you want to add a DOI manually or to request a DOI from the MPDL DOI service.

Change a Collection DOI

Note: You can only change a DOI if you already have assigned a DOI to the collection.

  • Go to Collections.
  • View the collection by clicking on its name.
  • Go to Action and click on Change collection DOI.

Search and Browse

The system offers two different searches, the simple search and the advanced search.


Simple Search

The Simple Search is displayed as a free text field on top of every page. Here the user can decide to either search for item, collection or album metadata. Search indexes and operators can be entered optionally to define more complex queries, e.g.

  • created>="2016-03-01" - to retrieve all items, collections or alben created at or after the specified date.
  • status="RELEASED" AND filetype="Document" - to retrieve all items of type document in state "released".

Please check the documention for further details on the query syntax.


Advanced Search

The Advanced Search is currently restricted to item metadata. First the user has to select the appropriate metadata profile. Afterwards the search scope can be restricted to a specific collection. Please choose a metadata element and enter or select the term you are looking for. Further search fields can be added via the plus button.


Browse

Click on Browse to display all items available to you.
