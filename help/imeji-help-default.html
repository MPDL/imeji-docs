Start with IMEJI


Register


Registration is free of charge. As the target group of our service are MPG employees, users with a MPG eMail account will get all required rigths automatically. All other users will only get very limited user rights with their registration. If further rights are required, please contact the support team.
  1. 1. Go to the registration page.
  2. 2. Fill in the form with at least your email, you family name and your organization.
  3. 3. Agree to the Terms of Use and click on Register now. You will get a confirmation message via e-mail.
  4. 4. Click on the confirmation link in the confiermation email. You will be forwarded to a confirmation page.
  5. 5. On the confirmation page, enter the password that was sent with the confirmation e-mail and logg in to the system.

Change Password


You have to be logged in.
  1. 1. Go to your user page by clicking on your user name on the top of the page.
  2. 2. Enter your new password twice (New password and Repeat password).
  3. 3. Click on Change password. Your new password is saved.

Forgot Password


Currently it is not possible to reset the passwort automatically. Therefore, please contact the support team.

Collections


Colections provide the main folder structure within IMEJI and are required to upload and organize items.
All available collections are listed below COLLECTIONS . If the user is not logged in, only released collections are visible. For logged in users, also private and discarded collections are visible. To get a better overview, the collections can be filterd via their state and their creator. When a collection is released, this is visible through the globe symbol, when it is shared via the people logo on the right side of the page.

Create Collection


Users require the appropriate authorization to create new collections. Normally, all users registered with a MPG e-mail account should have this authorization. For any questons regarding your account please contact the support team.
  1. 1. Go to the collections overview page and click on New collection.
  2. 2. Enter at least the required information (collection name, author name and organization). Information on further authors, a description of the collections content, aditional information (like a link to a website or publication) and a collection logo can be added optionally. Further on, an e-mail notification on item download can be activated.
  3. 3. Via the plus (and minus) buttons, further fields for authors and organizations can be added (or removed).
  4. 4. Click on Save.

Edit Collection Information


  1. 1. Go to the collection you want to edit.
  2. 2. In the menu Collection, click on Edit information.
  3. 3. Modify the values you wish to update and save them.

Share Collection


imeji offers three different types of sharing rights:
  1. Read: Permission to view and download the files, the metadata and the collection informations.
  2. Edit: In addition to the Read permission: uploading and deletion / discarding of files, editing of collection information, items and items license.
  3. Admin: In addition to the Edit permission: rights to share, release, discard and delete the collection.
  1. 1. Go to the collection you want to share.
  2. 2. In the menu Collection, click on Share.
  3. 3. Descide whether you want to share your collcetion with one or several individual users or with user groups:
    1. 3.1 Share with a user:
      1. 1. Enter the e-mail adresse(s) (separated by semicolon) of the users you want to share the collection with.
      2. 2. Select the permission of the user(s).
      3. 3. By activating Send e-mail the users will get an e-mail notification about the sharing.
      4. 4. Click on save.
      5. 5. If the choosen users are not registered within imeji, a message will occure where you can decide whether the new users should get an invitation for the usage of imeji or if you want to cancel the sharing process. When choosing the invitation, an invitation will be send via e-mail with registration instructions.
    2. 3.2 [If available] Share with a user group (This function is only availabe if at least one user group exists within imeji. User groups can be created by the system administrators.):
      1. 1. Select at least one user group from the list.
      2. 2. Select the permission of the user group.
      3. 3. By activating Send e-mail all users within the sepected user group(s) will get an e-mail notification about the sharing.
      4. 4. Click on save.

Release Collection


You can only release private collections. A release makes the collection and all its content permanently visible to the public. Afterwards neither the collection nor the items can be deleted any longer, they need to be discarded instead. But a release does not freeze the collection completely, in particular new items can still be added and metadata keeps editable, too.
  1. 1. Go to the collection you want to release.
  2. 2. In the menu Collection, click on Release.
  3. 3. A confirmation message appears which includes the information about how many items within the collection do not have a license assigned. This items will automatically get the license CC0 assigned (the defaut license of Edmond).
  4. 4. Click on release when that is fine with you. Otherwise click on cancel and do the license assignment manually (see Add license)

Delete / Discard Collection


Private collections can be deleted, published colections discarded. The diference between this two options is:
  1. Deleted collections will be vanished from the system without any traces.
  2. Discarded colections are collections, that already have been published and therefore have to be kept persistent. Discarded means, that the collection and all included items are not listed any more within the system via the standard search and browse funktionalities. But some basic metadata and the discard comment will be kept and displayed if the collection is requested via its URL or DOI. Further on, the collection administrator can still see the remaining information when filtering his collection list for discarded collections.
  1. 1. Go to the collection you want to delete / discard.
  2. 2. In the menu Collection, click on
    1. 1. [If the collection is private] Delete.
    2. 2. [If the collection is released] Discard.
    3. 3. Confirm the deletion. For discarding, a discard commeent wich should explain the discard has to be inserted bevore the discard can be confirmed.

Assign DOI to Collection


You can only assign a DOI to a released collection and you can only assign one DOI per collection. that means as soon as a collection already has an DOI, this functionality is not possible any more.
  1. 1. Go to the collection you want to assign a DOI to.
  2. 2. In the menu Collection, click on Add DOI to collection.
  3. 3. Accept the terms of use for the DOI assignment.
  4. 4. You can now cite your collection with the assigned DOI.

Cite collection


  1. 1. Go to the collection you want to cite.
  2. 2. Click on more, right next to the collection author.
  3. 3. Copy the text Cite as..

File Upload


  1. 1. Go to the upload page.
  2. 2. Choose a collection for the upload by selecting one (if you already have at least one collection) or create a new collection to upload your files.
  3. 3. When you choose an already published collection (visible through the globe in front of the collection name) you have to choose a license for the new file(s).
  4. 4. Select file(s) or drag and drop file(s) into the drop area.

Data quota exceeded

Per default, IMEJI users have a data quota from 1 GB. If this quota is used up, please contact:
  1. The support team if you are the creator of this collection.
  2. The owner of the collection, if the collection has been shared. You can find the owner of the collection on the share icon.

File format not allowed

For security reasons, IMEJI restricts the list of authorized file formats. To get more information, please contact the support team.

You do not have any permissions to upload items or create collections

Your account is not authorized to create a new collection, and you don't have permission to upload files in at least one collection. Please ask the support team for the authorization to create a new collection.

Describe Items


imeji offers you 3 different ways to describe your items with metadata:
  1. a. Single edit: Describe one item
  2. b. Batch edit: Describe several items of one collection
  3. c. Collection edit: Describe all items of one collection

Describe one Item

  1. 1. Go the item you want to enrich with metadata.
  2. 2. Click on Edit
  3. 3. Fill the form
  4. 4. [If you want to add a new metadata entry] Click on Add metadata:
    1. a. A list with all already existing metadata elements of the system will be displayed.
    2. b. Check if your desired metadata element already exists and if it is available, select it from the list.
    3. c. [If the metadata element is not available in the list] Click on Add metadata at the bottom of the list, create a new metadata element by defining a name (label), a type and optionally some predefined values and save the changes. Now the metadata element is added to the form of your item.
  5. 5. Click on Save.

Describe several Items of one Collection

  1. 1. Go to the collection containing the items you want to edit.
  2. 2. Select the items you want to edit.
  3. 3. In the menu X Selected Items, click on Edit.
  4. 4. The default metadata elements of IMEJI are displayed in columns (if nothing is displayed, there are no default metadata elements) together with the items names in rows.
  5. 5. Add a new metadata elements by clicking on Add Column.
    1. a. A list with all already existing metadata elements of the system will be displayed.
    2. b. Check if your desired metadata element already exists and if it is available, select it from the list.
    3. c. [If the metadata element is not available in the list] Click on Add metadata at the bottom of the list, create a new metadata element by defining a name (label), a type and optionally some predefined values and save the changes. Now the metadata element is added to the table.
  6. 6. Edit metadata values:
    1. a. Edit a single cell by clicking on to add a new value, or to remove a value.
    2. b. Add a value to a whole column, by clicking on and confirming via Add to all.
    3. c. Remove all values of a column, by clicking on at the top of the column.
    4. d. Remove a column, by clicking on at the top of the column.
    5. e. Change the name of column by clicking on at the top of the column.
  7. 7. Click on Save.

Describe all Items of one Collection

  1. 1. Go to the collection containing the items you want to edit.
  2. 2. [If some items are selected], click on Select none.
  3. 3. In the menu X Items, click on Edit all.
  4. 4. Click on Select metadata.
    1. a. A list with all already existing metadata elements of the system will be displayed.
    2. b. Check if your desired metadata element already exists and if it is available, select it from the list.
    3. c. [If the metadata element is not available in the list] Click on Add metadata at the bottom of the list, create a new metadata element by defining a name (label), a type and optionally some predefined values and save the changes. Now the metadata element is editable.
  5. 5. Enter a value.
  6. 6. Choose one method to add the metadata to the collection items:
    1. a. Add a new value to all: The value will be added to all items of the collection, no matter whether the items already have a value for this metadata element.
    2. b. Add only if value is empty: The value will only be added to the items of the collection which don't have any value for this metadata element yet.
    3. c. Overwrite all values: The value will be added to all items of the collection. Previous values for this metadata element will be removed.

Add License


  1. 1. Go the collection containing the items you want add the license.
  2. 2. Choose one the following options:
    1. a. Single edit: Go to the item itself and click on Edit. Togehter with all the other metadata, the license is displayed in the form and can be edited.
    2. b. Batch edit: Select some items and click on Edit all licenses in menu X Selected Items.
    3. c. Colletion edit: Click on Edit all licenses in menu X Items (ensure that no item is selected).
  3. 3. Enter a license:
    1. a. Select one predefined license from the list OR
    2. b. Click on Enter your own license and enter you license information.
  4. 4. Click on Save
    1. a. [If the collection is private] Choose if you want to replace the license of items that already have a license.
    2. b. [If the collection is released] The new license will be added to the item(s). Previous license will be discarded with a timestamp.

Delete / Discard Items


Items within private collections can be deleted, items within published colections discarded. The diference between this two options is:
  1. Deleted items will be vanished from the system without any traces.
  2. Discarded items are itmes, that already have been published and therefore could not vanish completely from the system. Discarded means, that the item itself is not available any more, but that its metadata is still displayed if the user is filtering the content of a collection for discarded items.
  1. 1. Go to the collection containing the items you want to delete / discard.
    1. a. [If the collection is private] Choose one the following option:
      1. i. Go to the page of the item and click on Delete.
      2. ii. Select some items and click on Delete in the menu X Selected Items.
      3. iii. Click on Delete all in the menu X Items (ensure that no item is selected).
    2. b. [If the collection is public] Choose one of the following options:
      1. i. Go to the page of the item and click on Discard.
      2. ii. Select some items and click on Discard items in the menu X Selected Items.
  2. 2. Confirm the deletion. For discarding, a discard commeent wich should explain the discard has to be inserted bevore the discard can be confirmed.

Move Items


You can only move item(s) from a private collection.
  1. 1. Go to the collection containing the items you want to move.
  2. 2. Choose one of the following options:
    1. a. Go to the page of the item and click on Move.
    2. b. Select some items and click on Move in the menu X Selected Items.
  3. 3. Select the collection you want to move the item(s) to from ths list of all collections you have at least editing rights.
  4. 4. [If the target collection is released] Select a license for the moved items and click on Move.

Download Items


There are different ways to download items:
  1. a. Go to the page of one item and click on Download file.
  2. b. Select one or several items wihin a collection, via Browse or via a Search and click on Download selected (Xfiles) in the menu.
  3. c. Go to one collection and click on Download (Xfiles) to download all items of the collection.

Browse and Search


Browse


Click on Browse to see all released items (together with all private items visible for you):

Simple Search


  1. The simple search allows to search for items (per default) or collections. During a search, all available metadata as well as the full texts (in case of text documents like pdf or doc) are included.
  2. When browsing within one collection, the simple search automatically only works for this collection.
  3. The simple search is compatible with the imeji search syntax.

Advanced Search


The advanced search is restricted to items and is (similar as the simple search) compatible with the imeji search syntax.

  1. 1. Go to the advanced search page.
  2. 2. Search for
    1. a. All fields (all available item metadata and optionally the fulltext).
    2. b. Specific metadata.
    3. c. File types (video, audio, docuument, image).
    4. d. Licenses.
    5. e. Technical metadata of the items.
  3. 3. Click on Search.
